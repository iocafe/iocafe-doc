Python
===================================================================
Python is a high-level, interpreted programming language known
for its simplicity and readability.

The iocafe packages use Python is used to run scripts which process
JSON files, copy files for specific micro controller builds, etc.

Python interface is compliled to iocompython, which is C extension
to provide Python API to iocom.
